what i do

Responsive Web-Application

web development approach that creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it.

Frontend Developer

Front-end web development, also known as client-side development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly.

Backend developer

Back end developers are responsible for creating and maintaining technology at the back end of a website (the server, database and application).

Clean Code

Clean code is code that is easy to understand and easy to change.This means the code is easy to read, whether that reader is the original author of the code or somebody else.

Software Maintenance

Software maintenance is a part of Software Development Life Cycle. Its main purpose is to modify and update software application after delivery to correct faults and to improve performance

Web Hosting

Web hosting is an online service that allows you to publish your website files onto the internet. So, anyone who has access to the internet has access to your website.
